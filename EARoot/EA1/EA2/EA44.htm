-- Definition -- <br/>A character string generally in the form of words of a language.<br/><br/>-- Description --<br/>The text data type is a combination of a string and a language identifier. It is useful for names and descriptions that are available in multiple languages. Where this is so, each version of the data should be included and each one associated with the relevant language identifier.<br/>
